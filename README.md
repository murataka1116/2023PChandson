# sample

## Overview
ポケモンWordleが Pythonで遊べるツールです。  
[@giga_yadoran](https://twitter.com/giga_yadoran)様による Web版は[こちら](https://wordle.mega-yadoran.jp/)。  

## Usage
```
python3 pokemon_wordle.py [-h] list
```

### Args
必須の引数は以下の通りです。

* list  
ポケモンのリスト (csvファイルのパス)
